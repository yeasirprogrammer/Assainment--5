Frequently Asked Questions

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a structured representation of a webpage that allows programming languages like JavaScript to interact with it. It represents the document as a tree of objects corresponding to elements, attributes, and text.

How can you select an element in the DOM?

JavaScript provides several methods to select elements in the DOM, such as:

  • getElementById("id"): Selects an element by its ID.
  • getElementsByClassName("class"): Selects elements by class.
  • querySelector("selector"): Selects the first matching element.
  • querySelectorAll("selector"): Selects all matching elements.
What is event delegation, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events from its child elements. This reduces the number of event listeners and makes it easier to manage dynamic content.

How can you manipulate an element's attributes and styles using the DOM?

You can use JavaScript to manipulate attributes and styles:

  • getAttribute(): Get an attribute's value.
  • setAttribute(): Set or change an attribute.
  • removeAttribute(): Remove an attribute.
  • style: Directly modify CSS styles, e.g., element.style.color = "red";
